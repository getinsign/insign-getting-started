Deploy This Demo

Run this demo on any of these platforms. The backend logic is shared across all variants.

Local (Node.js)

Run on your machine. Requires Node.js 18+.

git clone <repo-url>
cd sig-funnel
npm install
npm start
Glitch Free

Edit & run in the browser. Instant URL. No install.

Remix on Glitch

Or paste in Glitch: Import from GitHub → repo URL

StackBlitz Free

Runs Node.js inside your browser via WebContainers. Nothing to install.

Open in StackBlitz
Vercel Serverless

Deploys as serverless functions. Free tier. Set env vars in dashboard.

Deploy to Vercel 
INSIGN_URL=https://sandbox.test.getinsign.show
INSIGN_USER=controller
INSIGN_PASS=pwd.insign.sandbox.4561
Render Free

Free web service. Spins down after inactivity (~30s cold start).

Deploy on Render

Connect your GitHub repo, set start command: node src/server.js

Docker

Run anywhere with Docker. Self-hosted.

docker build -t sig-funnel .
docker run -p 3000:3000 sig-funnel
Shared Architecture

All platforms use the same core logic in src/routes.js. Express (server.js) and Vercel (api/*.js) are thin wrappers. No code is duplicated.
